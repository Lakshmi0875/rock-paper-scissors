# 🎮 Rock Paper Scissors Game

A beautiful, interactive Rock Paper Scissors game with an aesthetic design featuring animated backgrounds with paper folds, rocks, and scissors.

## ✨ Features

- **Aesthetic Design**: Beautiful gradient background with purple and pink tones
- **Animated Background**: 
  - Floating rocks, papers, and scissors with smooth animations
  - Paper fold effects that create a 3D perspective
  - Smooth sway and rotation animations
  
- **Interactive Gameplay**:
  - Three colorful buttons for Rock, Paper, and Scissors
  - Real-time score tracking
  - Instant win/lose/tie feedback
  - Smooth animations for choice display

- **User Experience**:
  - Responsive design (mobile, tablet, desktop)
  - Glassmorphism UI elements with backdrop blur
  - Hover effects and smooth transitions
  - Reset button to start a new game

## 🎯 How to Play

1. Click on one of the three buttons: **Rock**, **Paper**, or **Scissors**
2. The computer will make its choice
3. The winner is determined by the classic Rock Paper Scissors rules:
   - **Rock** beats **Scissors**
   - **Paper** beats **Rock**
   - **Scissors** beats **Paper**
4. Your score and the computer's score are updated automatically
5. Click **Reset Game** to start over
